# Privacy Policy

**Cluster Files** is built with user privacy as the absolute highest priority. 

## 🔒 Personal Data Collection
* **Zero Tracking**: We do not collect, monitor, store, or transmit any personal data.
* **No Telemetry**: There is no telemetry code, analytics suite, or usage tracking built into the app.
* **No Network Logs**: The application does not connect to external servers or send diagnostic reports.

## 📂 Local Permissions
All file access permissions are strictly used to perform standard local file operations (copying, moving, renaming, deleting, and archiving files) at the request of the user. No file metadata or directory indexing is ever uploaded.
